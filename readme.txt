Configuration de l'environnement :
python -m venv venv

Activation de l'environnement virtuel :
- macOS/Linux : source venv/bin/activate
- Windows : venv\Scripts\activate

Puis installez les dépendances :
pip install -r requirements.txt

Structure du projet :
- Chaque dossier correspond à une section d'exercices spécifique
- Les instructions pour chaque exercice se trouvent dans les fichiers instruction.txt
- Les fichiers d'exercices Python sont nommés exo_*.py
- Des ensembles de données d'exemple sont fournis pour les exercices d'analyse de données

Paquets requis :
- pyqt5 : Framework GUI
- scikit-learn : Bibliothèque d'apprentissage automatique
- numpy : Calcul numérique
- matplotlib : Visualisation de données
- pandas : Manipulation de données
- lime : Interprétabilité des modèles
- ipython : Shell Python interactif

Exécution des exercices :
1. Accédez au dossier de l'exercice
2. Exécutez : python exo_*.py
3. Suivez les instructions dans instructions.txt
